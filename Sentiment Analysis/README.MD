# Sentiment Analysis Dashboard
This project analyzes text data using Python and visualizes insights using Power BI.

## Tools Used
- Python (Pandas, NLTK, Matplotlib)
- Power BI
- NLP

## Steps Performed
- Data cleaning and preprocessing
- Sentiment analysis
- Visualization using Power BI
